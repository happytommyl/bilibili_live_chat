# README

## install dependencies

`uv sync`

## config

Create a `conf.ini` file that looks like this:

``` ini
[Credential]
SESSDATA = {your SESSDATA}
buvid3 = {your buvid3}
bili_jct = {your bili_jct}
DedeUserID = {your DedeUserID}
[Room]
roomid = {live_room id} 

```

loging into bili from a browser and get the fields from cache

## run

`python main.py`
